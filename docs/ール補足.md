# 業務ルール補足

## 案件番号採番の詳細

最新台帳の案件明細シートまたは案件追跡表から、
対象年月・担当者コードの最大連番を取得し +1 で採番する。

例：2603-1003-S01 が最大なら次は 2603-1004-S01

## 商品マスタ更新時の確認事項

1. PRDコード重複チェック（同一仕入先コード内の連番）
2. JANコードの桁数確認（8桁 or 13桁）
3. 仕入単価・販売単価は原則として既存値を保持（上書き指示がある場合のみ変更）

## PDF見積の読み取り方針

- pdfplumber でテキスト抽出を試みる
- 抽出失敗の場合はファイル名と抽出できた情報を報告し、手動入力を促す
- 金額は税抜・税込の区別を必ず確認・記録する

## ファイル保存タイミング

台帳に何らかの変更を加えた場合、作業完了時に必ず
`outputs/受発注管理台帳_YYYYMMDD_HHMM.xlsx` として保存する。
元の `data/raw/` ファイルは変更しない。
