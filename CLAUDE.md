# CLAUDE.md

## プロジェクト概要

新日本海商事（SNJ）の受発注管理台帳（SENDO v0.5）をClaude Code on the Webで操作するためのリポジトリ。

- **台帳本体**: `data/raw/受発注管理台帳_*.xlsx`（最新ファイルを使用）
- **入力ファイル**: `data/raw/`に投入（見積PDF、発注書Excel等）
- **出力先**: `outputs/`（タイムログ付きファイル名で保存）

---

## ファイル命名規則

### 台帳ファイル
```
受発注管理台帳_YYYYMMDD_HHMM.xlsx
例: 受発注管理台帳_20260317_1430.xlsx
```

### 出力CSV
```
[内容]_YYYYMMDD.csv
例: 商品マスタ_20260317.csv
例: 案件追跡表_20260317.csv
```

---

## シート構成（受発注管理台帳.xlsx）

| シート名 | 役割 | 操作 |
|---------|------|------|
| 商品マスタ | 商品・価格・JANコード管理 | 追加・更新 |
| 仕入先マスタ | 仕入先情報 | 参照 |
| 顧客与信管理 | 与信限度・残高 | 参照 |
| 案件明細 | 案件ごとの商品明細【主入力】 | 追加・更新 |
| 案件追跡表 | 進捗ステータス管理 | 更新 |
| 見積書 / 発注書 / 納品書 / 請求書 / 受領発注書 | 証憑テンプレート | 参照 |

---

## 採番ルール

### 案件番号
```
YYMM-1XXX-A01
例: 2603-1001-S01（2026年3月・杉田・大阪）
```

**担当者コード**: S=杉田 / T=田嶋 / H=蜂巣 / A=永井 / K=門永 / R=高山  
**支店コード**: 01=大阪 / 02=小樽

### 証憑番号
```
EST-SNJ-YYMM-1XXX-A01   （見積書）
POC-SNJ-YYMM-1XXX-A01-ZZZ  （発注書、ZZZ=仕入先コード）
DEL-SNJ-YYMM-1XXX-A01   （納品書）
INV-SNJ-YYMM-1XXX-A01   （請求書）
```

### 商品コード（PRD）
```
PRD_SNJ_[仕入先3文字]_[4桁連番]_[バリエーション2桁]
例: PRD_SNJ_NKH_0052_00
```

## 主なタスクと呼び出しキーワード

| キーワード | 処理内容 |
|-----------|---------|
| `商品マスタ更新` | 見積/発注書から商品情報を読み取り商品マスタに追記 |
| `案件番号採番` | 最新の連番を確認し次の案件番号を発行 |
| `明細記入` | 案件番号・商品コード・数量を指定して案件明細シートに追記 |
| `追跡表更新` | 案件番号とステータス・日付を指定して案件追跡表を更新 |
| `JAN参照` | 商品名またはPRDコードからJANコードを返す |
| `JAN抽出` | 商品マスタから条件指定でJANコード一覧をCSV出力 |
| `台帳保存` | 現在の台帳をタイムログ付きファイル名でoutputsに保存 |
| `CSV出力` | 指定シートをCSVとしてoutputsに出力 |

---

## 絶対ルール

- `data/raw/` のファイルは**読み取り専用**（直接編集しない）
- 台帳を更新した場合は必ずタイムログ付きで `outputs/` に保存
- 採番前に必ず既存の最大番号を確認してから次番を発行
- 仕入単価・販売単価の上書きは明示的な指示がある場合のみ実施
